# 京职搜

个人使用的北京招聘信息聚合网站第一版，覆盖互联网大厂、央企国企、事业单位和公务员四类职位。

## 已有功能

- 按关键词、招聘类型和学历筛选
- 按发布时间或报名截止时间排序
- 本机浏览器收藏职位
- 手机、平板和电脑自适应页面
- 所有职位保留原始信息来源链接

## 数据说明

当前 `app/page.tsx` 中是用于验证页面功能的示例职位。正式录入时，请为每条记录维护唯一 `id`、职位名称、单位、类别、地点、学历、招聘性质、发布日期、截止日期、标签、摘要、来源名称和官方原文链接。

建议优先使用国家公务员局、北京市人力资源和社会保障局、国聘平台和招聘单位官网。网站只保存检索所需摘要并跳转至原公告，不要高频访问来源网站。

## 本地运行

项目使用 Node.js 22 及 pnpm。安装依赖后执行 `pnpm dev`，生产构建执行 `pnpm build`。
