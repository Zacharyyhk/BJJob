# 数据源专用采集器计划

更新时间：2026-07-12

## 分类模型

一级分类固定为：

1. 互联网大厂
2. 北京市机关单位
3. 中央机关单位
4. 央国企

北京市机关单位和中央机关单位必须进一步标记：

- 事业编制
- 公务员编制

央国企和互联网大厂不使用编制类型字段。

## 每个专用采集器的验收标准

- 明确列表页、详情页、附件和分页边界。
- 保存原始字段，不因通用别名未命中而静默丢失。
- 只做来源内确定性的字段映射；语义匹配仍交给 Codex。
- 明确岗位 ID 稳定策略和技术型去重边界。
- 区分新公告、流程公告、拟聘公示和已过期内容。
- 使用真实页面样例增加解析测试。
- 运行增量准备脚本、语义分析和静态构建验证。
